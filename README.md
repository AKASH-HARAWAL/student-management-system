# Student Management System

A Full Stack Student Management System built using:

- Spring Boot
- MySQL
- REST APIs
- HTML/CSS/JavaScript
- Bootstrap
- Swagger
- JWT Authentication Basics

## Features

- Add Student
- Update Student
- Delete Student
- Search Students
- Export CSV
- Dark Mode
- Swagger API Documentation

## Technologies Used

- Java
- Spring Boot
- MySQL
- Bootstrap
- JavaScript
- Maven
